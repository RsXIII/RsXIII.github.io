Lazy Initialization: Η τιμή που επιστρέφει η μέθοδος, δεν

δημιουργείται μέχρι την πρώτη κλήση της. Αν επομένως ένα

μοναδιαίο αντικείμενο δεν χρησιμοποιηθεί, δεν δημιουργείται.

Στο παρακάτω πρόγραμμα, οι δύο εντολές printf επιστρέφουν την ίδια διεύθυνση για τους δείκτες A και A1.

int main() {

Accumulator* A;

A = Accumulator::getInstance();

Accumulator* A1;

A1 = Accumulator::getInstance();

printf("A address: %p\n", A);

printf("A1 address: %p\n", A1);

return 0; }
